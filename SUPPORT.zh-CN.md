如果你想使用、评估或贡献 `Scratch AI 教练`，建议按下面顺序找入口。

- 教师 Web 说明：[`apps/server-web/README.md`](apps/server-web/README.md)

## 什么时候提 issue

- Bug：请使用 `Bug Report`
- 功能建议：请使用 `Feature Request`
- 较大的方案讨论：先描述课堂场景、平台、用户类型和预期结果

## 提问时最好附上

- 平台：Windows / macOS / Linux / 其他
- 运行方式：`npm run server:dev`、`npm run server:api:dev`、`npm run server:web:dev` 或其他
- 是否配置了 DeepSeek Key
- 复现步骤
- 日志或截图

## 安全问题

不要公开提 issue，见 [`SECURITY.zh-CN.md`](SECURITY.zh-CN.md)。
