# Scratch AI 教练 / Scratch AI Coach

面向 Scratch 教学场景的开源服务器端工作区，维护基于 `FastAPI + Vue` 的教学链路。
An open source Scratch teaching server workspace built around the `FastAPI + Vue` classroom workflow.
