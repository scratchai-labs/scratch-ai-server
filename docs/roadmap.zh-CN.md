# 路线图

## 当前阶段

当前主线已经收口到服务器端教学版：

- 维护 `Go API + 教师管理 Web`
- 保持老师、学生、任务、提示和看板主链路可测试、可联调
- 保证新贡献者可以 clone、测试、构建和提 PR

## 下一阶段

- 继续补强自动化测试、部署文档和联调体验
- 完善 GitHub 开源协作体验
- 收口更清晰的部署、验证和发布节奏

## 未来计划

- 形成服务器端 + 学生客户端的稳定教学工作流
- 围绕 Scratch 课堂使用场景沉淀更完整的验证和示例项目
- 扩展课程、班级、历史记录和更细的课堂数据分析

## 暂不在本轮范围内

- 自动发布 GitHub Releases
- 大规模国际化产品改造
- 第一阶段不做课程 / 班级多层模型
- 第一阶段不做 WebSocket/SSE 实时推送
